✨ Funcionalidades
🔐 Autenticación (Login): Validación de credenciales en tiempo real con MockAPI y manejo de sesión con localStorage.
🛡️ Rutas Protegidas: Navigation Guards de Vue Router para restringir el acceso sin sesión.
📦 CRUD de Productos: Crear, editar y eliminar productos mediante modos conectados a MockAPI.
👥 CRUD de Usuarios: Gestión completa de usuarios con modales y alertas de Bootstrap.
🛒 Carrito de Compras: Añadir, quitar, vaciar y calcular totales en tiempo real.
🎨 UI/UX Premium: Efecto Glassmorphism en el login, diseño responsivo y animaciones fluidas.
🔔 Modales y Alertas: Confirmación visual para eliminar y alertas de éxito/error/warning.
🛠️ Tecnologías Utilizadas
Tecnología	Uso en el Proyecto
Vue.js 3	Principio del marco (API de composición)
Enrutador Vue	Enrutamiento y protección de rutas
Bootstrap 5	Estilos, grid, modales y alertas
Iconos Bootstrap	Íconos vectoriales
Axios	Cliente HTTP para la API
MockAPI	API REST falsa para el backend
Vite	Empaquetador y servidor de desarrollo
📂 Estructura del Proyecto
src/├── 📂 assets/             # Recursos estáticos├── 📂 components/         # Componentes (Navbar, Sidebar)├── 📂 router/             # Rutas y guardias├── 📂 services/           # Conexión MockAPI (ApiService.js)├── 📂 views/              # Vistas (Login, Productos, Usuarios)├── 📄 App.vue             # Componente raíz└── 📄 main.js             # Punto de entrada
💻 Instalación y Uso
Clona el repositorio:
intento

git clone https://github.com/esgomeza/examen-des-app-web.git
Ingrese a la carpeta del proyecto:
intento

cd examen-des-app-web
Instala las dependencias:
intento

npm install
Inicia el servidor:
intento

npm run dev
Abre tu navegador en 🌐http://localhost:5173
